# SS14 Brig Calc
> *Brig sentence calculator for Space Station 14 — Russian interface*

Калькулятор сроков заключения для игры **Space Station 14**. Помогает сотрудникам Службы Безопасности быстро рассчитывать сроки, учитывать модификаторы и формировать готовые приговоры прямо во время раунда.

 **[Открыть калькулятор](https://berber0s.github.io/ss14-brig-calc/)**

---

## Возможности

- **Таблица Корпоративного Закона** — все статьи от 1XX до 6XX с цветовым кодированием тяжести, выбирай кликом
- **Расчёт срока** — автоматическое суммирование с учётом правила 1.5× от максимальной категории
- **Смягчающие и отягчающие** — выбирай из списка или вводи свой процент вручную, с подсказками по каждому обстоятельству
- **Рецидив** — счётчик повторных нарушений с автоматическим +15% за каждое
- **Шкала тяжести** — визуальная полоска от зелёного до красного, показывает серьёзность приговора
- **Готовый бланк приговора** — одна кнопка копирует заполненный документ с перечислением статей, сроком и административным наказанием
- **Подсказки по статьям** — наведи на любую ячейку таблицы, чтобы увидеть описание и диапазон наказания

---

## Использование

Калькулятор работает прямо в браузере — никакой установки не нужно.

1. Открой [калькулятор](https://berber0s.github.io/ss14-brig-calc/)
2. Кликни по нужным статьям в таблице Корпоративного Закона
3. Настрой срок, смягчающие/отягчающие обстоятельства и рецидив для каждой статьи
4. Посмотри итоговый расчёт с детализацией по каждой статье
5. Нажми «Скопировать бланк» — получишь готовый приговор для вставки в игру

---

## Правила расчёта

- **1XX–5XX** — заключение с диапазоном сроков по категории
- **6XX** — пермабриг, лицо признаётся врагом корпорации
- **Суммирование** — статьи одной категории не складываются (берётся тяжелейшая), максимум = 1.5× от самой тяжёлой
- **Модификаторы** — применяются к каждой статье отдельно: смягчающие (5–50%), отягчающие (5–20%)
- **≥ 45 мин** → пермабриг, **≥ 70 мин** → пожизненное заключение
